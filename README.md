# to-do-pwa

> A Vue.js project

## Build Setup

``` bash
#1.
npm i vue-cli -g
#2.
vue init webpack-simple
#3.
npm i vuex --save
#4. 
npm install
#5.
npm run dev
#6.
npm run build
```

For detailed explanation on how things work, consult the [docs for vue-loader](http://vuejs.github.io/vue-loader).
